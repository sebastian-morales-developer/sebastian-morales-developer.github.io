My Applications

Application 1: Stoic Chatbot

Description: This application is a chatbot powered by GPT-4 and vector databases (pgvector) that helps users resolve personal challenges based on eight eudaimonic principles. These principles, organized by Sebastian Morales, are inspired by Stoicism and aim to improve quality of life. Users engage with a specialized AI agent offering philosophical insights and actionable guidance. It blends philosophy and technology seamlessly for self-improvement.

Stack Used: pgvector, OpenAI SDK, GPT-4 API, Flask, Python, PostgreSQL, Render.com for CI/CD.

GitHub repository: github.com/sebastian-morales-developer/stoic_chatbot_flask_1_public

Application 2: LangFaster - Learn Languages Faster

Description: This application powered by GPT-4 enables users to create monologues and conversations in over 58 languages, offering simultaneous results in both languages. It provides pronunciation guides based on the primary language, the International Phonetic Alphabet (IPA), and audio playback for both languages, facilitating faster and more effective language learning. The website is fully translated into 58 languages, making it accessible to a global audience.

Stack Used: OpenAI SDK, GPT-4 API, Ruby on Rails, Rails Internationalization (i18n), PostgreSQL, Render.com for CI/CD.

GitHub repository: github.com/sebastian-morales-developer/langfaster-app-public

Application 3: Books for Coding Learning

Description: This application powered by GPT-4 allows users to create comprehensive guides for learning to code, starting from the basics. It generates detailed documents packed with essential information for beginners to kickstart their coding journey.

Stack Used: OpenAI SDK, GPT-4 API, Node.js Express.js, Rails Internationalization (i18n), PostgreSQL, Render.com for CI/CD.

GitHub repository: github.com/sebastian-morales-developer/coding-books-nodejs-1-public
